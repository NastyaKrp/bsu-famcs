1) Парсинг : задача разбора и расчёта арифметических выражений в выполнена

2) Поддерживаемые архиваторы:
* zip
* jar

   Поддерживаемые шифровальные алгоритмы:
* симметричный алгоритм шифрования(ARS)


3) Вложенность:

Файл input может быть:
* только архивирован
* только зашифрован
* сперва архивирован, а потом зашифрован
* сперва зашифрован, а потом архивирован

Файл output также может быть:
* только архивирован
* только зашифрован
* сперва архивирован, а потом зашифрован
* сперва зашифрован, а потом архивирован

(выполнено в соответствии с поставленным условием)
